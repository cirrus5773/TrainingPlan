# Сайт тренировок

Русскоязычный статический сайт для плана на 2 месяца: тренировки, отметки выполнения, заметки, вес и дневной учет калорий.

## Что уже работает

- адаптация под телефон и ПК;
- автоматическая дневная норма калорий по выбранной неделе;
- добавление еды в течение дня и автоматический остаток калорий;
- отметки утренней и вечерней тренировки;
- заметки по дням;
- журнал веса;
- локальное хранение в браузере;
- синхронизация через бесплатный Supabase после настройки.

## Supabase

1. Создай бесплатный проект на Supabase.
2. Открой SQL Editor.
3. Выполни содержимое файла `supabase.sql`.
4. В Supabase открой Authentication -> Providers -> Email и включи email login.
5. В настройках сайта вставь Project URL и anon public key.
6. Введи email и нажми "Войти по email".

## GitHub Pages

В репозиторий нужно положить файлы:

- `index.html`
- `styles.css`
- `app.js`
- `supabase.sql`
- `README.md`

Потом в GitHub: Settings -> Pages -> Deploy from a branch -> main -> root.
